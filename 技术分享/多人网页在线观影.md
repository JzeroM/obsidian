### 介绍

#### FantasyPlayer是一个可多人网页在线同步观影的项目

#### 项目地址：[https://github.com/JzeroM/FantasyPlayer](https://github.com/JzeroM/FantasyPlayer)

#### FantasyPlayer的前端套用开源的DPlayer播放器，使用socket.io实现同步交互，使用nodejs作为后端。

### 部署

#### **服务器部署很简单：**

#### 第一步：打开服务器的3000端口

#### 第二步：服务器自行配置nodejs、npm和git环境

#### 第三步：git项目到服务器

```bash
git clone https://github.com/JzeroM/FantasyPlayer.git
```

#### 第四步：进入项目文件夹

```bash
cd FantasyPlayer
```

#### 第五步：下载项目使用的包

```bash
npm install
```

#### 第六步：启动后端服务

```bash
node server.js
```

#### 访问地址http://你的服务器公网IP:3000即可
